# Concesionario (Java)

Proyecto introductorio de programación orientada a objetos desarrollado en primer curso.

## Descripción

Aplicación en Java que modela un concesionario de vehículos utilizando herencia y colecciones.

Incluye:

- Clase base `Vehiculo`
- Subclases `VehiculoTurismo` y `Furgoneta`
- Clase `Concesionario` para gestionar el inventario

## Conceptos trabajados

- Programación orientada a objetos
- Herencia
- Encapsulación
- Uso de `ArrayList`

## Contexto

Proyecto académico inicial orientado a practicar fundamentos de POO en Java.
